# Exercício 1 – Função ou Procedimento

## Str()

Função.

Retorna uma string convertida a partir de um número.


## QOut()

Procedimento.

Apenas exibe uma mensagem na tela, sem retornar um valor útil.


## Date()

Função.

Retorna a data atual.


## Len()

Função.

Retorna o tamanho de uma string ou de um array.


## Upper()

Função.

Retorna o texto em letras maiúsculas.


## ClearScreen()

Procedimento.

Apenas limpa a tela, sem retornar um valor.
