# Exercício 4 – Refinamento Sucessivo

Neste exercício, o problema é dividido em uma visão geral (Nível 1) e, em seguida, cada etapa é detalhada (Nível 2).

## Problema

Um sistema de caixa de supermercado deve registrar os itens de uma compra, calcular o subtotal, aplicar desconto se o cliente tiver cartão fidelidade (5%) e mostrar o total a pagar.


## Nível 1 – Visão geral

1. Registrar os itens da compra.
2. Calcular o subtotal.
3. Aplicar desconto, se houver cartão fidelidade.
4. Mostrar o valor final da compra.


## Nível 2 – Detalhamento

### 2.1 Registrar os itens da compra

- Ler o código ou nome de cada produto.
- Registrar a quantidade.
- Registrar o preço de cada item.
- Repetir o processo até que todos os itens sejam registrados.

### 2.2 Calcular o subtotal

- Somar o valor de todos os produtos registrados.
- Armazenar o resultado como subtotal.

### 2.3 Aplicar desconto

- Verificar se o cliente possui cartão fidelidade.
- Se possuir, calcular 5% de desconto sobre o subtotal.
- Subtrair o desconto do subtotal.
- Caso contrário, manter o subtotal sem alterações.

### 2.4 Mostrar o valor final

- Exibir o subtotal.
- Exibir o desconto aplicado (se houver).
- Exibir o valor final a pagar.
